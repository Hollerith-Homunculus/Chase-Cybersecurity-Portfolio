# Windows Client Troubleshooting Lab

**Focus:** Help Desk / Service Desk Troubleshooting Practice

## Overview
Simulated common end-user issues in a Windows 11 VM to build Tier 1 support skills.

## Environment & Tools
- VirtualBox
- Windows 11 VM
- Tools: Event Viewer, Device Manager, ipconfig, PowerShell

## What I Did
- Created VM and installed Windows 11 evaluation.
- Simulated issues: no internet, driver errors, printer setup failure.
- Resolved using built-in tools and commands.
