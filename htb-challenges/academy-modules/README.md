# HTB Academy Modules

**Focus:** Foundational Cybersecurity Knowledge & SOC Analyst Preparation

## Overview
This subfolder documents completed Hack The Box Academy modules (free tier / beginner level).  
HTB Academy provides structured, hands-on learning in Linux/Windows fundamentals, networking, and security basics — directly reinforcing CompTIA A+ (OS/hardware) and Security+ (threats, vulnerabilities, incident handling) concepts.

Goal: Build practical understanding of systems, networks, and security principles to support entry-level IT Help Desk troubleshooting and Junior SOC alert triage.

- Browser: Chrome/Firefox (for interactive labs)
- Local tools (when applicable): VirtualBox (for following along with labs), Notepad++/VS Code for notes
- Modules completed (as of March 2026): Intro to the Academy, Network Fundamentals, Info-Sec, Linux Fundamentals, Windows Fundamentals
